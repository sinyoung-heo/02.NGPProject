Enter 세 번 누르면 바로 메뉴 선택창으로 이동 가능.

[ TILE EDIT ]
LEFT,RIGHT,UP,DOWN	: 화면 이동
숫자키 1~6			: 타일 이미지 변경.
S			: SAVE
L			: LOAD
F1 ~ F4			: 맵 이미지 변경

E			: 다시 메뉴로 돌아가기

타일 우클릭 이동으로 타일 이미지를 변경할 수 있습니다.

이미 저장되어있는 데이터가 있으므로 S키로 타일을 저장하지 말아주세요.

[ IN GAME KEY INPUT ]
1 	마을로 씬전환.
2 	상점으로 씬전환.
3 	듀토리얼 필드로 씬전환.
4 	중간보스로 씬전환
5 	보스로 씬전환.
6 	타일 Render On/Off.

W	Player 위로 이동.
A	Player 좌로 이동.
S	Player 뒤로 이동.
D	Player 우로 이동
R	Player 걷기/달리기 전환.
F1~F3	스킬등록.
F5~F7	포션.
SPACE	기본공격.
마우스 R	등록된 스킬 사용.


P	Player 정보창.
E	Player 장비창.
I	Player 인벤토리창.

L	경험치상승. -> Player 레벨업
----------------------------------
인벤토리에서 아이템 마우스 L버튼	: 아이템 장착.
장비창에서 아이템 마우스 L버튼	: 아이템 해제.

상점에서 아이템 L버튼		: 아이템 구매.
상점에서 인벤토리 아이템 L버튼	: 아이템 판매.
